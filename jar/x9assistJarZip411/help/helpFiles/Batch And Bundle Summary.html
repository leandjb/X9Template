The batch and bundle summary tab provides summary information for each batch (ACH) or bundle (X9) that is present within the currently loaded file. The batches tab is shown for ach files and includes specific columns based on that file format. Similarly the bundles tab is shown for x9 files and includes those columns that are appropriate when the file format is x9.37.

Error counters (error, warn, info) are provided which summarize the errors which were identified during validation. All columns are all sortable.
