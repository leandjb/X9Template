Tiff Tester is a convenience tool that allows you to load an external image for inspection and validation. Tiff Tester uses the same internal TIFF image validation rules within X9Assist that are applied to images which are embedded within x9 files. Tiff Tester allows you to quickly and easily perform these powerful validation rules against any TIFF image, even when the image is not currently embedded within an x9 file.

 

Before launching Tiff Tester from the tool bar, you must first ensure that the desired tiff rules are loaded for your image validation. These rules are loaded whenever an x9 file is processed. Tiff Tester will tell you which x9 rules (and hence tiff rules) are currently loaded on the status line in the lower left of the panel. You may need to first validate an x9 file (using your desired rules) to ensure that the correct TIFF rules are currently loaded.

 

Tiff Tester is very straight forward to use:

 

  • •Use the LOAD button to load your desired image. 

  • •The image will be displayed in the upper left. 

  • •The tiff tags associated with this image will be displayed in the upper right. 

  • •Any error messages that were identified during validation will be displayed at the bottom.
